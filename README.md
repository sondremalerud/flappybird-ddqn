# Flappy Bird DDQN 

## Overview

An implementation of Double Deep Q-Network (DDQN) in order to play Flappy Bird.

## Requirements
### Flappy Bird Gymnasium

```bash
pip install flappy-bird-gymnasium
```

### Other requirements

- Python 3.x
- Pygame
- Pytorch
- OpenAI Gymnasium
- Matplotlib
